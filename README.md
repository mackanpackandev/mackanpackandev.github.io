## Getting Started

Open `index.html` directly in a browser to view the site locally.

You can also use a local static server if you prefer:

```bash
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```

## Customization

Update the main content in `index.html`.

Common things to change:

- Page title
- Heading text
- Description text
- GitHub links
- Code sample

Update the visual design in `style.css`.

Common things to change:

- Colors in the `:root` variables
- Font sizing
- Spacing
- Button styles
- Responsive layout rules

## Deployment

This project is designed to work with GitHub Pages.

To deploy:

1. Push the files to the repository.
2. Open the repository settings on GitHub.
3. Go to **Pages**.
4. Choose the branch and folder GitHub Pages should publish from.
5. Save the settings.

Once deployed, the site should be available at:

```text
https://mackanpackandev.github.io
```
